Our Project

Purpose

The purpose is to implement a parser, that will get as an input code in the e language and will produce as an output a data structure representing the input code

Goals

Input: e source file

Output: Form tree, Global form table, Comment list

Form Tree

Each form contains additional source info.

It gives us more inforamtion about the source code.

Global form table

Global form table contains direct references to forms.

each one can be referenced by an index.

Comment List

saved all the single line comment in the source code.

It also save for each comment additional information likse start line, end line, position.
